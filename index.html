Country Helper

Welcome to Country Helper!

Country Helper is an interactive Bash (Bourne Again Shell) script that allows you to retrieve information about any country in the world and perform calculations for currency conversion.

Download

macOS Usage:

  1. Install the Homebrew package manager by pasting the following command into the terminal:
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    • If you already have Homebrew installed, you may skip to step 2
  2. Install the jq command-line processor using brew install jq
  3. To be able to run the Country Helper script, you will need to upgrade your bash shell. Do this by using brew install bash
  4. Close the terminal and open it again
  5. Navigate to the folder/directory where CountryHelper.sh is located
  6. Use chmod +x CountryHelper.sh or chmod 755 CountryHelper.sh to ensure that the script has executable permissions
  7. Run the script with the command bash CountryHelper.sh
  8. If you run into problems using the script, you may need to take some additional actions with the new bash shell you installed in step 3. Please refer to the following article for instructions: Upgrading Bash on macOS

Windows Usage:

If you do not have the Bash shell installed on your Windows machine, you will need to install it in order to run the Country Helper script. There are a few ways to do this and I recommend installing either Git Bash or Bash [on Ubuntu] on Windows

Using with Git Bash

  1. Open the Command Prompt and run as an administrator. You can do this by right-clicking on the Windows start button (or pressing Windows+X) and selecting Command Prompt (Admin)
  2. Install the Chocolatey Package Manager with the following command: @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
  3. Exit the Command Prompt and open it (running as an administrator) again
  4. Install the jq command-line-processor with chocolatey install jq
  5. Read and follow through with the instructions in this article to install Git Bash
  6. Open Git Bash
  7. Navigate to the folder/directory where CountryHelper.sh is located
  8. Use chmod +x CountryHelper.sh or chmod 755 CountryHelper.sh to ensure that the script has executable permissions
  9. Run the script with the command bash CountryHelper.sh

Using with Bash on Windows

  1. Open the Command Prompt and run as an administrator. You can do this by right-clicking on the Windows start button (or pressing Windows+X) and selecting Command Prompt (Admin)
  2. Install the Chocolatey Package Manager with the following command: @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
  3. Exit the Command Prompt and open it (running as an administrator) again
  4. Install the jq command-line-processor with chocolatey install jq
  5. Exit the Command Prompt
  6. Read through and follow the instructions in this article (up until you have finished setting your UNIX username and password)
  7. Run sudo apt update && sudo apt upgrade This will take a while. Be patient!
  8. You can now exit out of Ubuntu if you would like. As you will see in the final steps, we can use bash commands in Command Prompt with bash -c "command"
  9. Open the Command Prompt (just normally this time)
  10. Navigate to the folder/directory where CountryHelper.sh is located
  11. Use bash -c "chmod +x CountryHelper.sh" or bash -c "chmod 755 CountryHelper.sh" to ensure that the script has executable permissions
  12. Run the script with the command bash -c "bash CountryHelper.sh"

Linux Usage:

  1. Open the terminal
  2. Navigate to the folder/directory where CountryHelper.sh is located
  3. Use chmod +x CountryHelper.sh or chmod 755 CountryHelper.sh to ensure that the script has executable permissions
  4. Run the script with the command bash CountryHelper.sh
  • Note: The Country Helper script will detect your linux distribution and run the necessary command(s) to install the jq package manager [if not already installed]. However, if you prefer to install the jq package yourself, refer to the jq website.
