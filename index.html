macOS Usage:

  1. Install the Homebrew package manager by pasting the following command into the terminal: /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" (If you already have Homebrew installed, you may skip to step 2).
  2. Install the jq command-line processor using brew install jq
  3. To be able to run the Country Helper script, you will need to upgrade your bash shell.

Linux Usage:

  • Open the terminal
  • Navigate to the directory/folder where the CountryHelper.sh file is located
  • The script may or may not be executable upon download. Ensure that it has executable permissions by typing 'chmod +x CountryHelper.sh' or 'chmod 755 CountryHelper.sh in the terminal
